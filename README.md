## Description
Welcome to the Toronto Web Developer Site project! This project aims to showcase web development services and portfolio for a web developer based in Toronto.(https://project3-0.netlify.app/)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GSAP (GreenSock Animation Platform)

## How to Run

To run the this Website locally on your machine:

1. Clone the repository: `git clone https://github.com/your-username/project3.0.git`
2. Navigate to the project directory: `cd project3.0`
3. Open the `index.html` file in your web browser.
